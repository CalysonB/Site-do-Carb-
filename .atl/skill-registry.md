# Skill Registry - CARB-SITE-TESTE

## Project Standards

- **Backend**: Express.js with Sequelize. Use `server.js` as entry point.
- **Frontend**: Vite + React. Standard Vite project structure.
- **Linting**: ESLint is available in the frontend.
